asg0 displays a canvas and draws 2D vectors to it.\
Supports vector operations (add, div, etc.) and calculates vector values (angle between vectors, magnitude, etc.)
Uses the JavaScript library cuon-matrix-cse160 which is a modified library from the textbook "WebGL Programming Guide"
by Kouichi Matsuda and Rodger Lea.\
asg0.html: Describes layout of webpage, defines select boxes, input boxes, and buttons for vector input and operations\
asg0.js: Draws background and vectors to canvas based on input from document context. Also writes certain vector
operations to console. Uses the cuon-matrix-cse160 library to interface with vectors and complete vector operations.
