Croc is a cute little guy, but watch out, because if you poke him, he might bite you!\
This project supports camera rotation, panning, zooming, and a reset camera button.\
You can also control each of croc's limbs and his global rotation with the given sliders in the bottom left.\
The keyboard buttons 1-5 allow control of rendering, animation, and camera position.\
Shift + left click enables an alternative animation where croc bites at and shakes around his prey.
