Painting displays a canvas and has a variety of drawing features based on shapes.\
Supports modifying triangle vertices, rectangle side length, and circle vertex count.\
Uses the JavaScript library cuon-utils.js and webgl.utils.js which are libraries from the textbook\
"WebGL Programming Guide" by Kouichi Matsuda and Rodger Lea.\
Painting.html: Describes layout of webpage including sliders, tick boxes, buttons, and canvas.\
Painting.js: Reads shape sliders, draws shapes to canvas, includes "preview" function to see what you are\
going to place beforehand. Includes save functionality to save the current drawing to your hard drive as a JSON.\
Allows for both click to draw and hold to draw painting.
